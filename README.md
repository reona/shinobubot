## Usage


## Install

- cloneしてhubotを実行すると必要なものが勝手に入ります。

```
git clone https://github.com/reona/shinobubot.git
cd shinobubot
./bin/hubot
```

- slack に対応させる場合はアダプター名を指定して起動してください。(slack adapter はデフォルトでインストールされるようにしています。）

```
./bin/hubot --adapter slack
```
